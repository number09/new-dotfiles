# new-dotfiles

```
bash -c "$(curl -fsSL https://raw.githubusercontent.com/number09/new-dotfiles/main/install.sh)"
```


## 手動インストール
- 1password7(買い切り版使用)


## 手動設定
- bettertouchtool
    - apps内のプリセットをインポートする
- alfred
    - powerpackライセンス入力
    - preferences folderにapps/alfredフォルダを指定
- anyenv
    - anyenvに対してpyenv,nodenvをインストールする
